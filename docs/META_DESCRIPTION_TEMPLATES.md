# Quick Reference: Meta Description Templates

## Gold Rate Pages

### City Page Template
```
Check today's gold rate in {CITY} for 24K, 22K and 18K. Live gold and silver prices with daily updates and charts.
```

**Length:** ~120 characters
**Variables:** {CITY} = Delhi, Mumbai, Chennai, etc.

### Main Gold Rate Page
```
Check today's gold rate in all major Indian cities. Live 24K, 22K and 18K gold prices with daily updates and calculator.
```

**Length:** 130 characters

## Article/News Pages

### General News Template
```
{Brief summary of the article in one sentence}. Check latest updates and expert analysis.
```

**Length:** 140-160 characters
**Example:** "Gold prices surge 2% amid global uncertainty. Check latest city-wise rates and market analysis."

### Market Movement Template
```
{Asset} prices move today amid {reason}. Check latest rates, market reasons and expert insights.
```

**Example:** "Gold prices move today amid global cues. Check latest city-wise rates, market reasons and expert insights."

## Rules to Remember

### ✅ DO
- Keep length between 140-160 characters
- Use natural, human-readable sentences
- Include main keyword ONCE
- Provide clear value proposition
- Make each description unique

### ❌ DON'T
- Stuff keywords (e.g., "gold rate, gold price, today gold rate...")
- Use meta keywords tag
- Copy-paste same description for all cities
- Exceed 170 characters
- Use only commas without sentences

---

## Quick Checklist

Before publishing any page, verify:

- [ ] Meta description is 140-160 characters
- [ ] Description is a natural sentence
- [ ] Main keyword appears once
- [ ] No keyword stuffing
- [ ] Unique for this page (not duplicated)
- [ ] Provides clear value to user
- [ ] Open Graph tags are set
- [ ] Twitter Card tags are set
- [ ] Canonical URL is set
- [ ] JSON-LD structured data is present
